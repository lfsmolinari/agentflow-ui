# UX Stories

This folder contains UX-focused stories derived from the phase 1 product requirements and the current layout direction inspired by Codex Desktop.

These stories are not implementation stories. They define the user flow, screen behavior, and interaction expectations that a specialized UI/UX agent can turn into visual and interaction specifications for Architect and Coder.

## UX Direction

- Minimal, desktop-first workspace experience
- Dark, quiet, low-distraction interface inspired by Codex Desktop
- Left rail for workspace and session context
- Large central chat area as the primary focus
- Agent selection inside the composer, only when a session is active
- Phase 1 optimized for Strategist workflows, while hinting at future multi-agent expansion
